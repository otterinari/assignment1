2. Difference between internet and intranet

Internet Intranet
  1. The Internet is a globally-connected network of computers that enables people to share information and communicate with each other.
  2. In internet, there are multiple users.
  3. Internet is unsafe.
  4. Internet is a public network.
  5. Anyone can access Internet.
  6. Internet provides unlimited information.
  1. An intranet,is a local or restricted network that enables people to store, organize, and share information within an organization.
  2. In intranet, there are limited users.
  3. Intranet is safe.
  4. Intranet is a private network.
  5. Intranet can be accessed only by the users of the organisation.
  6. Intranet provides limited information.
